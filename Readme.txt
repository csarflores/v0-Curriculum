# 📌 Curriculum Vitae con IA V0 de Vercel

Este proyecto fue desarrollado completamente con la inteligencia artificial V0 de Vercel, utilizando tecnologías modernas como **Next.js** y **Tailwind CSS**. Mi aporte principal consistió en ajustar algunos detalles visuales y personalizar la información para adaptarla a mis necesidades.

## 🚀 Tecnologías utilizadas

- **Next.js**: Framework de React para aplicaciones web rápidas y escalables.
- **Tailwind CSS**: Framework de estilos para una interfaz moderna y responsive.
- **V0 de Vercel**: Plataforma de inteligencia artificial utilizada para generar la estructura base del proyecto.
- **Netlify**: Plataforma de despliegue que permite hosting eficiente y optimizado.

## 🎯 Objetivo del Proyecto

El propósito de este proyecto fue explorar las capacidades de la inteligencia artificial en el desarrollo web, creando un currículum vitae con un diseño atractivo y moderno. Además, me permitió evaluar hasta qué punto la IA puede optimizar y agilizar el trabajo de los desarrolladores frontend, reduciendo tiempos de producción y mejorando la eficiencia en la creación de interfaces.

## 📌 Cómo Ejecutarlo Localmente

Si deseas clonar y ejecutar este proyecto en tu entorno local, sigue estos pasos:

1. Clona el repositorio:
   ```bash
   git clone https://github.com/tuusuario/nombre-del-repo.git
   ```
2. Accede al directorio del proyecto:
   ```bash
   cd nombre-del-repo
   ```
3. Instala las dependencias:
   ```bash
   npm install
   ```
4. Inicia el servidor de desarrollo:
   ```bash
   npm run dev
   ```
5. Abre el navegador y accede a `http://localhost:3000`
